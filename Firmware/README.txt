    Microcontroller firmware was developed using the Microchip MPLAB IDE v8. A
freeware copy of this software, as well as additional information, can be found
at http://microchip.com/mplab. Firmware is written in the C programming language
and compiled using Microchip's C18 compiler. For a freeware version of this
compiler, as well as additional information, see http://www.microchip.com/c18
    CPLD firmware was developed using Xilinx ISE Design Suite v11. A freeware
copy of this software, as well as additional information, can be found at
http://www.xilinx.com/products/design-tools/ise-design-suite/index.htm. CPLD
firmware is written in Verilog.
